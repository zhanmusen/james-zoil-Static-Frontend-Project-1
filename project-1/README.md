# Static Front-End Website

This project is a simple three-page static website built using **HTML5** and **CSS3**. It demonstrates clean structure, semantic markup, and a consistent navigation system.

## Features

- Three fully functional pages: Home, About, Contact
- Custom CSS styling
- Responsive layout structure
- Clear navigation menu
- Clean, readable code

## Value to Users

This website provides a lightweight, fast-loading, and easy-to-navigate experience. It serves as a template for beginners learning front-end development or for anyone needing a simple static site structure.

## Technologies Used

- HTML5
- CSS3
